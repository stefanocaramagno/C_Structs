**Hello everyone!**

My name is [Stefano Caramagno](https://www.linkedin.com/in/stefanocaramagno), and I'm pleased to present this repository containing exercises on **structs** implemented in **C programming**. 
These exercises were completed as part of the **Foundations of Computer Science** course during my [Bachelor's Degree in Computer Engineering](https://www.dieei.unict.it/corsi/l-8-inf) at the [University of Catania](https://www.unict.it).

This repository hosts a collection of exercises focused on using structs in C programming. 
The exercises cover defining, initializing, and manipulating structs to model complex data structures.

## üéØ Objectives

- **Struct Definition**: Learn to define and initialize structs to represent complex data.
- **Data Organization**: Organize data effectively using structs, enhancing code readability and maintainability.
- **Basic Struct Operations**: Understand basic operations with structs, such as accessing and modifying fields.

- **Struct Basics**: Exercises on defining and initializing structs.
